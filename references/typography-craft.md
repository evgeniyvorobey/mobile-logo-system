# Typography Craft — Runtime Checklist

> Full guide with classification details, pairing examples, and case strategy: [typography-craft-guide.md](typography-craft-guide.md)

## Key Rules

### Classification
Know the family before choosing: geometric sans, humanist sans, neo-grotesque, slab serif, modern serif (didone), display/custom. Each has a character, best pairings, and risk.

### Pairing Logic
Two approaches:
- **Contrast**: symbol and wordmark from different families (geometric symbol + humanist type). Creates visual interest.
- **Harmony**: symbol and wordmark share structural logic (same stroke widths, corner radii). Creates unity.

### Weight Matching
Visual weight (stroke thickness, counter size, density) must match between symbol and wordmark. Squint test: should feel like they belong together.

### Size Ratio (cap-height to symbol height)
- 100%: equal partnership
- 60-70%: symbol-dominant (most common for app logos)
- 40-50%: symbol primary, wordmark subordinate

### Kerning
Default kerning is for body text, not logo scale. Evaluate every pair manually:
- round-round: tighten slightly
- straight-straight: standard
- open pairs (LA, VA, TA): tighten significantly
- problem pairs (rn, cl): check for merge at small size

### Lockups
- **Horizontal**: symbol left, wordmark right. Gap: 50-100% of cap-height.
- **Stacked**: symbol above, wordmark below, centered. Gap: 30-60% of cap-height.
- **Clear space**: cap-height on all four sides. Nothing enters this zone.

### Case
lowercase = friendly/modern. Title Case = professional default. UPPERCASE = bold/authoritative. Choose intentionally.

### Customization
Safe: modify terminal, create ligature, modify dot on i/j, adjust ascender/descender.
Dangerous: change letter proportions, remove recognizable features.
Rule: visible at medium size, invisible at small size.

## Checklist

- [ ] Type classification identified and justified
- [ ] Pairing logic stated (contrast vs harmony)
- [ ] Visual weight matches symbol
- [ ] Size ratio defined
- [ ] Every letter pair kerned at logo scale
- [ ] Tracking justified (tight / standard / loose)
- [ ] Horizontal lockup with measured gap
- [ ] Stacked lockup with measured gap (if needed)
- [ ] Clear space defined
- [ ] Case strategy chosen and justified
- [ ] Tested at minimum legible size (~12-14px rendered)
- [ ] Tested in icon lockup at 60px and 29px (if in icon)
- [ ] Customizations tested at small size
- [ ] Font license verified
