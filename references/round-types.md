# Round Types

Use this file to decide whether flat, material, and monochrome rounds are necessary.

## Flat Round

Run a flat round when:
- the mark is new and core shape strength is still unproven
- the concept depends on gradients or glow
- Android monochrome/themed icon support matters
- the user asks for a cleaner, stricter form

Flat round goals:
- simplify the silhouette
- reduce primitive count
- prove shape memory without surface tricks

## Material / Premium Round

Run a material or premium round when:
- the product truly benefits from crafted tactility
- premium perception is a real brand goal
- the flat mark is already strong

Material round goals:
- add surface richness without harming the silhouette
- keep the outer form stable
- avoid noisy luxury clichés

Do not run this round before the flat core is credible.

## Monochrome Round

Run a monochrome round when:
- Android themed icons are in scope
- the mark uses contrast between fills heavily
- the user wants packaging for real mobile release

Monochrome round goals:
- preserve identity with one color
- remove unnecessary internal complexity
- keep weight balanced after color removal

## Wordmark Round

Run a wordmark round when:
- the selected symbol is stable
- the user wants a logo system, not symbol-only output

Wordmark goals:
- choose an appropriate typographic voice
- define lockups
- ensure the symbol and wordmark do not compete

## Rule Of Order

Default order:
1. concept round
2. selection
3. refinement
4. flat
5. material if needed
6. monochrome
7. wordmark pairing
8. package

Only break this order when the user explicitly asks for a later-stage artifact first.
