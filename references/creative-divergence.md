# Creative Divergence — Runtime Checklist

Use this before concept territories when the user asks for more variety,
stronger creativity, a less generic mark, a reset, or a category where icons
usually look interchangeable.

The goal is broader idea range without decorative noise.

## Divergence Axes

Explore difference across at least four axes before choosing territories:

- **Metaphor class**: literal product cue, symbolic promise, system/process,
  counter-category metaphor, abstract signature shape
- **Silhouette topology**: container, cut-out, monogram, directional form,
  modular system, negative-space form, compound mark
- **Construction system**: circle grid, square module, keyline, root rectangle,
  gesture-then-rationalized, mono-first Android grid
- **Emotional temperature**: precise, warm, playful, premium, technical,
  editorial, energetic, calm
- **Platform priority**: iOS-first, Android adaptive-first, monochrome-first,
  store-listing-first, notification-small-first
- **Category distance**: familiar, category edge, category break

## Thumbnail Pass

Before the first scored concept round, create 8-12 quick seeds as text, SVG,
or Pencil frames.

Each seed should be one concise idea:
- name
- changed axis or axes
- core silhouette
- why it is not just a color/style variant
- primary risk

Do not count these as distinct seeds:
- same shape with a different gradient
- same symbol with softer corners
- same mark with a small cutout
- same metaphor rendered in another style

## Shortlist Rules

Shortlist 3-5 territories from the seeds:
- one project-aligned / evolutionary
- one adjacent
- one category-distinctive
- one stretch, if the brief allows
- one monochrome-first, if Android themed icons matter

Reject seeds that cannot explain `29px` survival, one-color mapping, or
platform adaptation.

## Novelty Triage

Use a quick 1-5 pre-score before the main evaluation matrix:

| Seed | Brand fit | Mobile survival | Novelty | Cliche risk | Production risk |
|------|-----------|-----------------|---------|-------------|-----------------|

Advance seeds with a useful tension: high novelty plus acceptable brand fit,
or strong brand fit plus a clear distinctiveness move.

## Cliche Inversion

List the 3 most likely category cliches, then define the intentional avoidance
move for each.

Example:
- Cliche: generic leaf
- Avoidance move: use cyclic rhythm or layered timing instead of botanical shape

## Worked Examples

Use these examples to calibrate the difference between true range and decorative
pseudo-variety.

### Bad Set: Decorative Pseudo-Difference

Brief: personal finance tracker that feels generic.

| Seed | Claimed difference | Why it is rejected |
|------|--------------------|--------------------|
| Blue Vault | blue gradient, rounded container | Same vault metaphor and same container silhouette |
| Green Vault | green palette, softer corners | Only color and radius changed |
| Neon Vault | brighter glow and highlight | Rendering effect, not a new idea |
| Vault With Coin | small internal coin cutout | Same topology plus fragile 29px detail |
| Vault Dark Mode | black/gold premium version | Style variant, same monochrome mapping |

Diagnosis:
- all seeds share the same metaphor class: literal finance/security
- all seeds share the same silhouette topology: rounded container
- all seeds use color/effects as the main difference
- the monochrome version would collapse into one generic safe/vault shape

Action: keep only the strongest vault variant as one evolutionary option, then
force new seeds across different axes.

### Good Set: True Divergence

Same brief: personal finance tracker that feels generic.

| Seed | Axes changed | Why it advances | Main risk |
|------|--------------|-----------------|-----------|
| Ledger Fold | metaphor: record-keeping; topology: folded plane; grid: root rectangle | Feels financial without using a coin, card, chart, or vault | May feel like a documents app |
| Signal Reserve | metaphor: control signal; topology: directional mark; grid: 12-module square | More distinctive in a row of blue finance icons | Could drift toward analytics |
| Pocket Compass | metaphor: decision guidance; topology: negative-space pointer; platform: monochrome-first | Strong one-color Android path and clear 29px cue | Could feel navigation-related |
| Quiet Stack | metaphor: accumulated progress; topology: modular stack; tone: calm/utilitarian | Easy to adapt into wordmark rhythm and app UI geometry | May be too restrained |
| Split Balance | metaphor: tension and balance; topology: asymmetric counterform; category distance: edge | Avoids category cliches while preserving finance logic | Needs careful geometry to avoid crypto cues |

Why this set is stronger:
- each option changes at least four axes
- each has a different silhouette memory
- each has a plausible one-color mapping
- the risks are different, so the evaluation can make a real choice

## Pseudo-Difference Filter

Before presenting the concept set, compare every pair of seeds.

If two seeds share four or more of these, they are probably variants, not
separate territories:
- same metaphor class
- same outer silhouette
- same construction system
- same reduction rule
- same monochrome mapping
- same platform adaptation
- same emotional tone

When pseudo-difference appears:
1. merge the weaker seed into the stronger seed as a refinement note
2. create a replacement seed by changing the most repeated axis
3. re-run novelty triage before scoring

Useful replacement moves:
- shift from literal object to symbolic promise
- shift from filled container to negative-space form
- shift from iOS-first surface treatment to Android monochrome-first silhouette
- shift from category-familiar to category-edge while preserving brand fit
- shift from static object to system/process metaphor

## Output

Before presenting concepts, summarize:
- axes explored
- seeds rejected and why
- shortlisted territories
- strongest creative risk
- safest fallback

If time is tight, compress the thumbnail pass into a brief divergence snapshot,
but do not skip the range check when the user explicitly asks for more creative
or more varied work.
