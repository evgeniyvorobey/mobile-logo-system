# Premium Craft — Runtime Checklist

> Full guide with rationale and examples: [premium-craft-guide.md](premium-craft-guide.md)

## Key Rules

- Negative space is designed, not leftover. Fill counterforms with color — they must look deliberate.
- Fill ratio: 40-60% typical. Outside this range needs explicit justification.
- Single light direction for all material effects. No mixed shadows.
- Depth should be felt, not seen (0.5-2% opacity inner shadows, 3-5% luminance gradients).
- Edge treatment consistent around entire mark.
- No texture at icon size. Exception: 1-3% opacity grain at 120px+, invisible at 60px.
- Outer silhouette simple and memorable. Interior rewards closer inspection.
- Max 3 colors in the mark. More reads as consumer, not premium.
- Single focal point. If eye goes "everywhere," mark lacks hierarchy.
- Gradient: one direction, two stops, 10-25% luminance range, must have flat fallback.
- Detail budget: 3-5 elements max in the mark.
- Stroke terminals: all round, all flat, or all angled — never mixed.
- Corner radii from a single scale (e.g., r, r/2, r×2).
- Weight distribution intentional (centered=stability, top-heavy=authority, etc.).
- Curve junctions: G1 (tangent continuity) minimum, G2 preferred for organic shapes.
- Clean vector paths: minimum anchor points, no overlapping paths.

## Longevity

- Identity must be structural (shape, proportion, negative space), not stylistic (gradient trend, shadow trend).
- Style-agnostic silhouette test: render as (1) flat fill, (2) outline, (3) cut-out. Must be recognizable in all three.
- 5-year durability: would this look dated in a retrospective?

## Anti-Patterns

Metallic textures, excessive bevel, drop shadows as primary depth, glossy bubbles, gradient banding, ornamental borders, sparkles/lens flares/glow as identity, decorative detail in silhouette that clogs at small size, effects only for high-DPI.

## Checklist

- [ ] Counterforms pass "fill the negative space" test
- [ ] Fill ratio 40-60% (or justified)
- [ ] Single consistent light direction
- [ ] Depth effects subtle (felt, not seen)
- [ ] Edge treatment consistent
- [ ] Outer silhouette simple and memorable
- [ ] Interior rewards inspection without being required
- [ ] Color count <= 3
- [ ] Single focal point
- [ ] Gradient restrained with flat fallback (if used)
- [ ] Detail budget: 3-5 elements max
- [ ] Stroke terminals consistent
- [ ] Corner radii from single scale
- [ ] Weight distribution intentional
- [ ] Curve junctions G1+
- [ ] Paths clean (minimum anchors)
- [ ] Style-agnostic silhouette test passed
- [ ] 5-year durability answered
- [ ] No anti-patterns present
